# Lenguaje de Marcas - Xtart Manager

## Descripción
Este proyecto consiste en el desarrollo de la interfaz web del sistema **Xtart Manager**, una aplicación de gestión de ligas de fútbol.

La página web ha sido desarrollada utilizando **HTML** para la estructura del contenido, **CSS** para el diseño visual y **JavaScript** para añadir interactividad y dinamismo.

El objetivo del proyecto es aplicar correctamente los principios del Lenguaje de Marcas, incluyendo:

- Uso de etiquetas semánticas en HTML
- Separación de estructura y presentación
- Implementación de `media queries` para diseño responsive

## Instrucciones para abrir y visualizar la web
1. Descargar o clonar el repositorio.
2. Acceder a la carpeta del proyecto.
3. Abrir el archivo `index.html` en cualquier navegador  (Chrome, Firefox, Edge...).
4. Navegar entre las distintas páginas mediante el menú de navegación.

## Diseño responsive

El diseño se adapta a dos tamaños principales de pantalla mediante el uso de media queries, `1024px` para tablet y `600px` para móvil. Se ha ajustado:

- Distribución de columnas
- Tamaño de fuentes
- Organización del footer
- Tamaño de elementos 

Entre otras funcionalidades que se han considerado necesarias a la hora de desarrollar el contenido.
